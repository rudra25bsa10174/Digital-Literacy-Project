**Project Overview**
This repository contains my Digital Literacy Portfolio Project, completed as part of the CSE0001 course. 
As a Student Digital Ambassador, the goal of this project is to promote responsible, safe, and professional use of digital tools among students.
The project covers five modules, each focusing on a key aspect of digital literacy including communication, online presence, collaboration tools, and cybersecurity awareness.
